## Content

### performance-evaluation.ipynb and llm-based-analysis.ipynb

The python scripts used to run the LLM-based data extraction from case report on interactions between radiotherapy and the systemic therapies Pembrolizumab, Cetuximab and Cisplatin.


### data/
The .JSON file used in the experiments containing the data structure with topics and categories for the data-extraction tasks.
The .CSV files containing the titles and abstracts of the case reports.
